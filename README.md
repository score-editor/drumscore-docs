# DrumScore Editor Documentation

Official documentation and help content for [DrumScore Editor](https://github.com/score-editor/drumscore-editor) - the music notation application for pipe band drumming.

## About

This repository contains user-facing documentation, tutorials, and help content for DrumScore Editor (DSE). Documentation is available in multiple languages to serve our international pipe band community.

## Available Languages

- English
- French
- German


## For Users

If you're looking for help with DrumScore Editor:
- Browse the documentation in your preferred language
- Visit [drumscore.scot](https://drumscore.scot) for downloads and support
- Contact support: alan@drumscore.scot
